Welcome to the skeet shooter of the future!

The object of the game is to destroy all the discs before the timer runs out. Each disc moves at a constant speed, but you can't tell one from another. Click a square before the next refresh to fire.

Enter in custom game settings (or use the default ones), click START, then fire away!

Try a big range with a short refresh rate for a visual treat!
